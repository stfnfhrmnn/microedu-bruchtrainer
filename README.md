# Bruchtrainer

MVP learning app for fraction multiplication and division (Klasse 6, Hamburg). The product specification lives in `coding_agent_prompt_bruchtrainer_app_stufe_1_2.md`.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```

`npm test` runs unit tests (Vitest) plus task verification (`verify-tasks`).
